# FocusFlow - Chrome Productivity Extension

## Overview
FocusFlow is a Chrome productivity extension designed to help users stay focused and manage their daily tasks efficiently.

## Features
- Task Management
- Pomodoro Timer
- Daily Goal Tracker
- Notes Section
- Website Blocker
- Theme Switcher
- Productivity Analytics
- Custom Logo

## Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extension API
- Chrome Storage API

## How It Works
Users can add tasks, track completed tasks, set daily goals, write notes, and use a Pomodoro timer for better focus. It also blocks distracting websites like YouTube, Instagram, and Facebook.

## Installation
1. Download the project
2. Open Chrome
3. Go to chrome://extensions/
4. Enable Developer Mode
5. Click Load Unpacked
6. Select project folder
