# 🌦 Weather App

A modern weather application built using **React.js**, **Node.js**, **Express.js**, and **OpenWeather API**.

## 🚀 Features
- Search weather by city
- Real-time weather updates
- Temperature details
- Feels like temperature
- Humidity information
- Wind speed
- Weather condition icons
- Loading indicator
- Error handling
- Responsive modern UI

## 🛠 Technologies Used
- React.js
- Node.js
- Express.js
- Axios
- OpenWeather API
- CSS3
